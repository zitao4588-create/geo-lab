# AI曝光体检 H5 推广方案

日期：2026-07-05
状态：草案 v1（对应线上 release `20260705175108`）

## 0. 方案前提：四个硬约束

任何推广动作都建立在这四个约束之上，不满足就降级或推迟：

1. **合规口径**：备案子域名复核完成前，`exposure.playgamelab.cn` 只以「内测邀请 / 个人项目分享」口径传播。不投流、不发商业硬广、付费服务不进 H5、不公开报价。（依据 DECISIONS.md 2026-07-03 决策）
2. **容量上限**：全局 30 份报告/天、1 份/IP/小时。推广目标不是冲流量，而是**用高意向用户填满每天 30 个名额**。名额用完的 429 文案已引导扫码，本身就是转化路径。
3. **证据边界**：只有 DeepSeek 单平台真实采样。所有素材禁用「保证 AI 排名/推荐提升」「全网 AI 曝光」类表述；引用用户报告需本人授权；不虚构案例和评价。
4. **转化闭环**：推广开始前必须补齐 `VITE_CONSULT_WECHAT_ID`（复制微信号）；建议同步加最轻量的来源追踪（见 §6），否则渠道效果无法归因。

## 1. 定位与信息屋

- **主张（campaign line）**：顾客问 AI 时，会提到你吗？
- **产品一句话**：30 秒填资料，拿 20 个真实问题去问 DeepSeek，生成一份带原始答案证据的 AI 曝光体检报告，免费。
- **差异点（对外反复讲）**：
  1. 真实采样：不是问卷打分，是真的调 DeepSeek API 提问并留档原文；
  2. 证据边界：每条结论标注来源，不夸大、不承诺排名——「诚实」本身是差异化；
  3. 稀缺诚实叙事：每天只放 30 个免费名额，因为每份报告都是真实采样、真实成本。

三个内容角度（所有素材从这三个里长出来）：

| 角度 | 情绪 | 典型标题 |
|---|---|---|
| 扎心测试 | 焦虑/好奇 | 「我把自己的小程序放进 AI 里问了 20 个问题，提及率只有 40%」 |
| 趋势教育 | 认知升级 | 「AI 时代的 SEO 叫 GEO：小商家现在该做的 3 件事」 |
| 诚实工艺 | 信任 | 「为什么我的免费体检每天只放 30 个名额」 |

## 2. 三阶段推进

### 阶段 0：补闭环（1-3 天，推广前置条件）

- [ ] 配置 `VITE_CONSULT_WECHAT_ID` 并重新构建部署（5 分钟）。
- [ ] 加 `?from=` 来源参数记录（见 §6 P0 项）。
- [ ] 启动备案口径复核（登录 ICP/公安备案后台或问接入商）——不阻塞阶段 1，但阻塞阶段 2。
- [ ] 朋友圈 + 2~3 个熟人微信群小范围试水（10~20 人），验证「填写完成率」和「加微率」，修文案。

### 阶段 1：内测口径的内容播种（第 1-4 周）

**口径**：独立开发者 build in public + 内测邀请。不是广告，是「我做了个东西 + 我的真实数据」。

| 渠道 | 打法 | 频率 |
|---|---|---|
| 朋友圈/微信群 | 首发 + 每周一条进展（数据、用户反馈、翻车记录） | 2-3 条/周 |
| 公众号 | 冰箱小雷达 before/after 深度案例长文打底，再按三个角度各写一篇 | 1 篇/周 |
| 知乎 | 回答「GEO 是什么」「AI 搜索怎么优化」「DeepSeek 会推荐什么」类问题，经验贴口吻，文末内测邀请 | 2-3 答/周 |
| 即刻/开发者社区 | build in public 流水账：上线、限流设计、采样成本、UI 焕新过程都是素材 | 跟随节奏 |
| 小红书 | 深色报告封面截图做封面图，「你的店 AI 根本不知道」类标题试 3-5 篇看数据 | 2 篇/周 |

**阶段 1 目标（两周后校准）**：
- 每天 5-15 份真实报告（名额利用率 20-50%）；
- 每周 3-5 个加微咨询；
- 拿到 3 份可公开引用的用户反馈（经授权）；
- 记录各渠道 `from` 参数分布，砍掉无效渠道。

### 阶段 2：合规确认后的放大（第 5 周起，双前提）

前提 A：备案口径复核通过；前提 B：第二采样平台接入（报告说服力升级）。

- 短视频（抖音/视频号）：30-45 秒屏录脚本量产（见 §4）；
- 异业合作：小程序开发服务商、建站工作室、私域操盘手——他们把体检报告当作自己提案的敲门砖（B2B2B 转介绍）；
- 行业垂直群渗透：餐饮老板群、医美运营群、SaaS 创始人群，用对应行业的匿名化样例报告开场；
- 评估小额投流（此时才允许）；
- GEO 自证叙事升级：公开发布「我们给自己做 GEO 的月度复测曲线」。

## 3. GEO 自证：最强的信任资产（贯穿全程）

吃自己的狗粮：给「AI曝光体检」本身做 GEO——上品牌页/FAQ/对比页，把产品自己的 20 题 Prompt Universe 固定下来做月度复测，公开发布提及率变化曲线。

- 这同时是：产品能力证明 + 每月一篇天然内容 + 服务交付流程的彩排。
- 文案示例：「我们不承诺让 AI 推荐你，但我们把自己从 X% 提到了 Y%，方法和证据全公开。」

## 4. 首批素材草稿

**朋友圈首发**：
> 做了个小工具：AI曝光体检。你的顾客已经开始问 DeepSeek「XX哪家好」了,但 AI 的答案里未必有你。填 30 秒资料,它会真的拿 20 个问题去问 DeepSeek,把原始回答和一份 GEO 分析报告给你,免费。每天只放 30 个名额——每份报告都是真实采样,是真金白银的 API 成本 😂 内测中,想测的点链接或私我。

**知乎回答骨架**（用于「GEO/AI 搜索优化」类问题）：
1. 先给亲历数据钩子（冰箱小雷达 20 题提及率 40%，附截图）；
2. 讲方法论：诊断 → 优化 → 发布 → 监测，强调证据边界和「别信保证排名的」；
3. 结尾轻邀请：「做了个免费自测工具，内测中，需要的评论区/私信」。

**公众号选题池**：
1. 顾客问 AI「哪家好」的时候，你的店在答案里吗？
2. 我把自己的小程序放进 AI 里问了 20 个问题（冰箱小雷达实测）
3. GEO 不是玄学：一份真实采样报告长什么样
4. 为什么我的免费体检每天只放 30 个名额
5. 被 AI 误读比被 AI 忽略更可怕：聊聊风险信号

**短视频 30-45s 脚本**：
1. （屏录）问 DeepSeek：「西安家政哪家靠谱？」→ 镜头扫过答案：没有你；
2. 字幕：「你的顾客每天都在这么问」；
3. （屏录）打开 H5 → 30 秒填表 → 报告封面得分揭晓（深色封面就是视觉钩子）；
4. 收尾：「免费，每天 30 个名额，真实采样不注水。链接在评论区。」

**小红书**：封面用深色报告封面截图；标题试「你的店，AI 根本不知道」「30 秒查你的品牌会不会被 AI 推荐」「开店的人 2026 年该懂点 GEO」；正文放 3 张报告截图 + 边界声明。

## 5. 漏斗与复盘节奏

漏斗定义：访问 → 开始填写 → 提交成功 → 查看报告 → 扫码加微 → 咨询 → 成交人工体检。

- 初期基准全部未知，**前两周只做校准**，不设 KPI；
- 每周五复盘一次：`submissions.jsonl` 计数 + `from` 分布 + 加微人数（手工记）+ 各平台后台数据；
- 名额连续 3 天打满 → 是提高全局限额（成本决策）或收紧口径的信号；
- 单渠道两周零转化 → 停更该渠道。
