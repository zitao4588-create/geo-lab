# 全物料配图方案 + GPT Image 出图提示词

## 使用说明（先读）

1. **两类配图，边界不能混**：
   - **真实截图类**（报告数据、产品界面）→ 必须用线上真实页面截图，**禁止 AI 生成**——生成假报告图等于伪造证据，踩项目红线。
   - **概念插画类**（氛围、隐喻、封面底图）→ 用下面的提示词生成。
2. **所有插画提示词都设计为「画面无文字」**：GPT Image 渲染中文容易错字，标题一律后期用设计工具（稿定/Canva/Keynote）压字，或直接用已生成的 `posters/*.png`。
3. 每条提示词末尾都带同一段「风格锚」，保证全套物料视觉统一；生成时建议一次出 2-4 张挑选。
4. 通用负向要求（每条都带）：不要出现任何真实品牌 logo、不要水印、不要写实人脸、不要文字。

**风格锚（已内嵌进每条提示词，也可单独复用）**：
> 扁平矢量插画风格，深蓝夜色渐变背景（从 #0D1730 到 #0A1122），主色品牌蓝 #2E5CE6 与亮蓝 #4A7DFF，点缀琥珀金 #E9BC74，大量留白，构图简洁现代，有轻微胶片噪点质感，高级感

## 生成参数建议

- 模型：GPT Image（gpt-image 系列最新版），质量选 high
- 尺寸：横图 1536×1024，竖图 1024×1536，方图 1024×1024（GPT Image 原生三档，比例不合适的后期裁切）
- 每条 prompt 出 2-4 张；挑选标准：留白是否够、金色问号是否醒目、有没有意外出现文字/乱字（有就重roll或裁掉）
- 全套图生成后放进 `marketing/posters/generated/`，文件名对应编号（W1.png、X2.png……）
