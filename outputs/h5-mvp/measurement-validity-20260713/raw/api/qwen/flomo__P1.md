## Raw answer

flomo（浮墨笔记）是一款专注于轻量级记录的个人笔记工具。它的核心理念是帮助用户像发微博或朋友圈一样，快速捕捉碎片化的想法和灵感，而不必担心排版、分类或复杂的整理工作。

其主要特点包括：
1. **极简输入**：支持多平台（Web、iOS、Android、微信服务号等）快速记录，强调无压力的记录体验。
2. **标签管理**：不使用传统的文件夹层级结构，而是通过标签（Tag）来组织和关联笔记。
3. **回顾机制**：提供每日回顾等功能，帮助用户重新发现过去记录的内容，促进知识的内化。
4. **数据隐私**：官方宣称注重用户数据隐私，并提供数据导出功能。

它适合需要频繁记录短文本、灵感、待办事项或阅读摘录的用户，不适合用于撰写长篇文章或进行复杂的项目管理。
