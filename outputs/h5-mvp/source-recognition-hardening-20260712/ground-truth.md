# 官方来源与实体范围 Ground Truth

核验日期：2026-07-12

## 判定口径

- `entity_matched`：页面标题或描述能够与目标品牌/实体建立直接关系。
- `scope matched`：页面还能证明当前产品、型号、城市或门店层级。
- `scope partial`：页面是官方相关入口，但不能证明用户提交的更窄范围。
- `scope mismatched`：页面属于相关品牌，但层级不足或与目标范围不一致。
- 页面存在、HTTP 200 或属于同一域名，均不能单独证明具体实体事实。

## P01：松下 ES-LM55

- 官方入口：`https://consumer.panasonic.cn/product/beauty-health/mens-shaver-grooming/shaver/ES-LM55.html`
- 预期来源关系：`entity_matched`
- 预期范围关系：`matched`
- 可证明：松下大锤子2.0、电动剃须刀、型号 ES-LM55、五刀头、磁悬浮马达、IPX7 防水、约 55 分钟续航。
- 不自动证明：实际推荐效果、市场排名、用户口碑或未出现在页面中的承诺。

## P02 / E03：用户型号与官方页面冲突

- 使用同一松下 ES-LM55 官方入口。
- 官方来源仍应识别为有效产品页，不能因为用户填写 ES-LV9C 就把来源整体降为无效。
- 页面主实体型号为 ES-LM55；用户填写 ES-LV9C 时必须记录事实冲突或错误认知。
- 页面“同类产品”区域出现 ES-LV9C，不能把它误当成当前页面主型号。

## L01：海底捞门店搜索入口

- 官方入口：`https://www.haidilao.com/serve/storeSearch`
- 页面 HTML 标题和官方域名能够建立海底捞实体关系。
- 当前无需 JavaScript 的抓取结果不能证明西安具体门店、地址、服务半径或预约状态。
- 预期来源关系：`entity_matched`
- 预期范围关系：`partial`
- 结果应说明“已识别相关官方入口，但仍缺少西安具体门店证明”，不得把入口本身当成具体门店事实。

## L03：全国品牌首页

- 输入入口：`https://www.haidilao.com/`
- 可以证明品牌存在，但不能证明西安具体门店服务。
- 预期来源关系：`entity_matched`
- 预期范围关系：`mismatched`
- 必须暂停具体门店层级的正确实体识别和总分。

## E02：无关 URL

- 天气、资讯或其他与目标实体没有标题/描述关系的页面不得绑定为官方实体来源。
- 预期来源关系：`unrelated`
- 预期范围关系：`mismatched`
- 不得支持正确实体识别或总分。

## L04：未证明的服务承诺

- 页面或 URL 存在不能证明“百分百除菌”“全城最低价”“绝对安全”。
- 这些内容只能保留在待核验事实中，不得进入已确认事实。
