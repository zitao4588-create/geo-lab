目标：为 AI 曝光体检 H5 建立一个默认关闭、可独立验证的公开网页搜索层，接好 AnySearch、Tavily、Jina Search 和火山方舟 Web Search，并通过固定 canary 判断哪个服务适合免费 H5 的候选来源发现；搜索结果不得被解释为消费端曝光或已验证事实。

执行方式：先复用 `outputs/anysearch-benchmark-20260707/` 的既有证据，再实现统一后端 adapter、环境变量模板、隔离 canary runner 和本地假响应测试。不要接入 `/api/diagnoses`、不要部署、不要把密钥放进仓库、不要自动转为付费。真实外部 canary 前先报告 4 个 provider × 12 条查询的 48 次请求上限、各 provider 免费额度和可能的模型 Token 费用，并等待确认。

成功标准：四个 adapter 均能在无真实外部调用的测试中正确构造请求、标准化 URL/标题/摘要、处理缺失密钥/超时/错误并避免泄露凭据；12 条查询和评分规则固定；canary 原始响应、评分和选择结论有隔离目录；未改变生产诊断、限流、报告口径和 `sources/`。

输出格式：说明新增和修改文件、四个 provider 的配置状态、测试结果、真实 canary 尚需的密钥/开通项、费用风险，以及下一步唯一动作。
