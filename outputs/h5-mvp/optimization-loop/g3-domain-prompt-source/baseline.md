# G3 基线

记录时间：2026-07-12

修改前：

- 实体商品问题硬编码胡须、刀头、防水与续航，不能泛化到空气净化器等商品。
- 软件问题默认“微信里”，缺少主体、输入输出、限制和替代方案。
- 本地服务问题仍出现合同、企业式资质和售后模板，缺少距离、门店、营业时间、排队、预约、价格、停车、过敏与儿童服务。
- PageAudit 直接自动跟随 `fetch`：没有私网/DNS rebinding、跳转、MIME、响应大小、流式限长或循环跳转防护。
- 来源目标没有内容哈希、命中片段、抓取方式和时效状态；动态页面没有受控渲染接口。
- 候选 URL 只有泛化 `candidate` 状态，没有独立的本地待审核接口。

首个 G3 测试因 `validateAuditUrl` 尚未导出而失败；这固定了安全与 Prompt 改造前的红灯基线。
